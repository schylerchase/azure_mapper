Azure Mapper Guide

        
      

      

        
Getting Started

        
1. Run Azure CLI commands (listed in the left sidebar) and paste JSON output into the matching text areas

        
2. Click Render Map to visualize your infrastructure

        
3. The Account field at top labels data from different Azure subscriptions.

Save and Load

        
Save downloads your workspace as a .azuremap file. Open loads it back. Compatible between web and desktop. Auto-saved to browser every 30s.

        
Compliance

        
Click the Compliance chip in the stats bar to open the full dashboard. Checks include CIS Benchmarks, Well-Architected Framework, RBAC analysis, and Azure Architecture Best Practices. Mute individual findings, filter by severity/framework, export as CSV.

        
Interactions

        
Click a VNet to lock highlight. Click a Subnet for NSG rules. Click a Gateway to highlight routes. Hover for previews.

        
Design Mode

        
Press D to enter. Use the palette to add VNets, subnets, gateways, and resources. Press E to export a plan with CLI commands.

        
Layouts

        
Grid — VNets side by side. Landing Zone — Hub-spoke topology. Executive — Simplified overview.
